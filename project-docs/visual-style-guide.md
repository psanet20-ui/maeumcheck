# 오늘의 마음체크 비주얼 스타일 가이드

## 한 줄 방향

`오늘의 마음체크`는 병원 도구처럼 보여서는 안 됩니다.  
이 앱은 `검사 앱`보다 `따뜻한 담임선생님의 마음 공책`처럼 느껴져야 합니다.

핵심 인상은 아래 세 가지입니다.

- 안전하다
- 부드럽다
- 조용히 위로한다

## 기본 무드

브랜드 무드는 `위로 한 스푼`입니다.

- 햇살이 드는 교실 창가
- 오래 쓴 종이 공책
- 비 오는 날의 조용한 우산
- 정원이 있는 작은 집
- 따뜻한 차 한 잔 옆에 둔 기록장

즉, 차갑고 디지털적인 의료 서비스보다 `사람의 온기가 남아 있는 생활 위로` 쪽이 맞습니다.

## 아이콘 방향

아이콘은 너무 기술적이거나 날카로우면 안 됩니다.

추천 방향:

- 라운드형 선 아이콘
- 선 두께는 얇지도 두껍지도 않게 `중간`
- 모서리는 둥글게
- 내부 디테일은 최소화
- 한눈에 뜻이 보이는 단순한 형태

추천 아이콘 톤:

- 날씨: 해, 구름, 비, 무지개
- 회복: 잎, 새싹, 햇빛, 물방울
- 기록: 공책, 연필, 메모 카드
- 연결: 편지, 말풍선, 우산, 손
- 안정: 집, 창문, 난로, 정원

피해야 할 것:

- 병원 십자가, 심전도, 경고 삼각형 남발
- 과하게 입체적인 3D 아이콘
- 네온 컬러 아이콘
- 금융 앱 같은 딱딱한 라인 아이콘
- 지나치게 귀여운 유아용 캐릭터 느낌

## 일러스트 방향

일러스트는 설명용보다 `감정 동반자`처럼 보여야 합니다.

추천 방향:

- 작은 집, 창문, 정원, 구름, 햇빛 같은 상징 요소
- 사람 얼굴은 과장하지 않고 단순한 실루엣 위주
- 직접적인 눈물 표현보다 날씨나 공간으로 감정을 비유
- 여백이 넉넉한 반평면 스타일
- 채도는 낮고 명도는 부드럽게

좋은 예시 분위기:

- 비가 오지만 창문에 작은 불빛이 켜진 집
- 구름 사이로 햇빛이 비치는 하늘
- 새싹이 올라오는 화분
- 의자, 스탠드, 머그컵이 놓인 조용한 책상

피해야 할 것:

- 우는 얼굴 클로즈업
- 너무 만화적인 표정 과장
- 슬픔을 직접 자극하는 어두운 연출
- 공포 영화 같은 그림자
- SNS 스티커처럼 가벼운 밈 스타일

## 캐릭터 방향

이 앱의 대표 캐릭터가 필요하다면 `작은 새싹` 또는 `작은 집`이 가장 잘 맞습니다.

### 캐릭터 후보 1: 작은 새싹

- 감정 회복과 성장의 상징
- 부담 없이 귀엽게 쓸 수 있음
- 마음종례, 감성유형과도 연결 쉬움

표현 방식:

- 눈, 입은 아주 작게
- 지나치게 활발하지 않게
- 조용히 응원하는 표정

### 캐릭터 후보 2: 마음의 집

- 앱의 대표 기능과 직접 연결됨
- 상태 변화 표현이 쉬움
- 브랜드 차별성이 큼

표현 방식:

- 창문 불빛, 지붕, 작은 나무로 감정 표현
- 얼굴처럼 보이지 않아도 됨
- 날씨와 연결하면 확장성이 좋음

추천 우선순위:

1. 메인 상징은 `마음의 집`
2. 보조 캐릭터는 `작은 새싹`

## 색 사용 원칙

색은 감정을 세게 밀어붙이지 말고 `살짝 번지는 느낌`이 좋습니다.

기본 계열:

- 크림
- 아이보리
- 베이지
- 살구
- 세이지 그린
- 비구름 블루
- 장밋빛 회색

사용 원칙:

- 배경은 늘 밝고 숨 쉴 공간이 있게
- 상태 변화는 색조 차이로 표현
- 위험 상태만 분명한 붉은 기운 사용
- 검정 대신 진한 갈색 계열 사용

## 화면 요소별 추천 그래픽

### 1. 첫 화면

- 햇살이 드는 종이 카드
- 구름, 새싹, 창문 같은 장식 요소 소량
- 시선을 뺏지 않는 부드러운 배경

### 2. 마음 날씨

- 아이콘 중심
- 작은 날씨 배지 느낌
- 선택 시 색이 번지는 카드 효과

### 3. 마음의 집

- 대표 일러스트 영역
- 점수에 따라 집 상태가 바뀌는 구조
- 비, 창문 불빛, 정원, 난로 같은 요소 사용

### 4. 위로 카드

- 종이 카드 또는 편지 카드 느낌
- 한쪽 모서리에 작은 잎사귀나 해 아이콘
- 텍스트가 가장 잘 보이게 장식은 절제

### 5. 회복 차트

- 차가운 데이터 차트처럼 보이지 않게
- 연필로 그은 듯한 선 느낌
- 점 포인트는 둥글고 부드럽게
